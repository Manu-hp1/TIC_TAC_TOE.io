Tic Tac Toe
in C++

A console-based Tic Tac Toe game implemented using C++ and Object-Oriented Programming. Demonstrates class design, game logic, board validation, and win-condition handling.

Source Code Watch Demo

Game Features

Two Player Mode

Supports Player X and Player O with turn-based logic.

Win Detection

Checks rows, columns, and diagonals for winning conditions.

Draw Handling

Identifies draw when all cells are filled without winner.

Core Implementation Concepts

  • Class-based game structure
  • 2D array board representation
  • Input validation
  • Loop-based game engine
  • Clear console UI design

Technology Stack

C++ OOPS Arrays Game Logic Console Application
